# GAME DESIGN DOCUMENT: Sea Battle Tactical // UCUP-2026

**Назва проєкту:** Sea Battle: Tactical  
**Жанр:** Тактичний симулятор / Покрокова стратегія  
**Сеттінг:** Мінімалістичний військовий інтерфейс (Cyber-Naval)

---

## 1. Концепція (Core Idea)
Модернізація класичної настільної гри «Морський бій». Проєкт перетворює знайому механіку на динамічне тактичне протистояння за рахунок введення спеціальних здібностей та симетричного штучного інтелекту. Візуальний стиль фокусується на атмосфері сучасного командного містка з преміальним, мінімалістичним UI/UX.

## 2. Core Loop (Ігровий цикл)
1. **Стратегічна підготовка:** Розміщення флоту з дотриманням суворих регламентів.
2. **Тактичне виявлення:** Використання радара для звуження зони пошуку.
3. **Вогневе ураження:** Комбінування звичайних пострілів та точкових авіаударів.
4. **Аналіз та адаптація:** Моніторинг бойових логів для вирахування стратегії AI.

## 3. Механіки та Системи
### 3.1. Класична база
* Поле 10х10 клітинок.
* Склад флоту (1х4, 2х3, 3х2, 4х1) згідно з офіційним регламентом UCUP-2026.
* Сувора перевірка колізій (кораблі не торкаються навіть по діагоналі).

### 3.2. Тактичні здібності (Твіки)
* **Air Strike (Авіаудар):** Ураження зони 3х3 клітинки. Дозволяє швидко зачищати сектори, але має ліміт (1 запуск на місію).
* **Radar Sweep (Радар):** Миттєве сканування зони 2x2. Повідомляє про наявність ворожих палуб, не завдаючи шкоди.

### 3.3. Симетричний Smart AI
Штучний інтелект не просто стріляє рандомно. Він використовує систему **Target Queue** (черга цілей) для добивання поранених кораблів та має ідентичні до гравця ресурси (Авіаудар), що забезпечує справедливий і складний геймплей.

### 3.4. Game Feel (Соковитість)
* **Screen Shake:** Трясіння екрана при отриманні пошкоджень.
* **Particle System:** Динамічний розліт часток («іскор») при влучанні.
* **Haptic Feedback:** Вібровідгук на критичні події.

## 4. Технічні особливості
* **True Responsive:** Адаптивна верстка, що підтримує будь-яке співвідношення сторін вікна.
* **Combat Logs:** Повна система фіксації подій з можливістю миттєвого експорту бойового звіту.
