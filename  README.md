# Sea Battle: Tactical // UCUP-2026 Game Jam

### **Про проєкт**
Тактична реалізація гри «Морський бій» на фреймворку **Flutter**. Проєкт розроблений студентом 1-го курсу ННІ «Каразінський банківський інститут» ХНУ ім. В. Н. Каразіна.

Основна мета — продемонструвати поєднання високої інженерної культури (SOLID) та сучасного ігрового дизайну.

### **Технічний стек**
* **Engine:** Flutter (Desktop SDK)
* **State Management:** Provider (архітектурний розподіл Logic/UI)
* **Audio:** Audioplayers (багатоканальний звук)
* **VFX:** Custom Tween Animations & Particle Systems
* **CI/CD:** GitHub Actions (автоматизований білд під Windows Win64)

### **Як запустити (Win64)**
1. Завантажте архів `sea_battle_tactical_win64.zip` з розділу релізів/артефактів.
2. Розпакуйте вміст у будь-яку папку.
3. Запустіть файл `sea_battle.exe`.
   *Примітка: папка `data` та файли `.dll` мають знаходитися в одній директорії з виконуваним файлом.*

### **Особливості для оцінювання (Checklist)**
* **QA Compliance (25%):** Алгоритм розміщення виключає будь-яке торкання кораблів (включаючи діагональне).
* **ENG Excellence (15%):** Код повністю декомпонований. Логіка гри (`GameEngine`) відокремлена від віджетів візуалізації. Відсутні «магічні числа», реалізована чітка система іменування.
* **GX & Game Feel (15%):** Реалізовано трясіння камери, анімації часток та звуковий резонанс.
* **System Features:** Інтегрований тактичний термінал (кнопка іконки в кутку) з функцією **EXPORT**, що копіює всі логи бою в буфер обміну для звітності.
